
# Fintech Wallet Backend (Full)
This package contains a fully implemented backend scaffold for a Fintech digital wallet using TypeScript, Express, Sequelize (Postgres), Passport (JWT), Swagger (OpenAPI), Nodemailer and Twilio placeholders.

## Quick start
1. Copy `.env.example` to `.env` and set values.
2. `npm install`
3. `npm run dev` (development)
4. Swagger UI: http://localhost:3000/docs
